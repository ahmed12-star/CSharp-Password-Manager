# 🛡️ C# Password Manager

A secure and lightweight command-line application built with **C#** and **.NET** to manage and store passwords locally with custom encryption.

## 🚀 Features
* **Secure Storage:** Saves credentials in an external file.
* **Custom Encryption:** Uses a substitution cipher logic for basic data protection.
* **Full CRUD Operations:** Add, List, Get, and Delete passwords easily.
* **File Insights:** View metadata like file size and creation time directly from the app.
* **Interactive UI:** Uses console colors for better user experience and feedback.

## 🛠️ Technologies & Concepts
* **Language:** C#
* **Framework:** .NET 8.0
* **Data Structures:** Dictionary (for fast lookups).
* **Concepts:** File I/O, StringBuilder, Enums, and Exception Handling.

## 📖 How to Use
1. Clone the repository.
2. Open the solution in **Visual Studio**.
3. Run the project and follow the on-screen menu instructions.
